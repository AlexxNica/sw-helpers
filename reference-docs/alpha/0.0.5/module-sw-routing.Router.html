Router

The Router takes a set of Route's and will direct fetch events to those Route in the order they are registered.

Instance Methods

this.registerRoute(input)

Registers a route with the router.

Parameters
input Object
Properties
route Route

The route to register.

Register routes will take an array of Routes to register with the router.

Parameters

Properties
routes Array.<Route>

this.setCatchHandler(input)

If a Route throws an error while handling a request, this catch handler will be called to return an error case.

Parameters
input Object
Properties
handler Handler

A handler to deal with errors in routes.

this.setDefaultHandler(input)

A default handler will have it's handle method called when a request doesn't have a matching route.

Parameters
input Object
Properties
handler Handler

A handler to deal with default routes.
