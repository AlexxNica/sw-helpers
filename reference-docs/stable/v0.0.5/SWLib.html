SWLib

This is a high level library to help with using service worker precaching and run time caching.

Constructor

new SWLib()

Initialises the classes router wrapper.

Members

this.router

A getter for the Router Wrapper.

Instance Methods

this.cacheRevisionedAssets(revisionedFiles)

If there are assets that are revisioned, they can be cached intelligently during the install (i.e. old files are cleared from the cache, new files are added tot he cache and unchanged files are left as is).

Parameters
revisionedFiles Array.<String>

A set of urls to cache when the service worker is installed.

this.setOptions(options)

Can be used to define debug logging, default cache name etc.

Parameters
options Object

The options to set.

this.warmRuntimeCache(unrevisionedFiles)

If there are assets that should be cached on the install step but aren't revisioned, you can cache them here.

Parameters
unrevisionedFiles Array.<String>

A set of urls to cache when the service worker is installed.
