RouterWrapper

A simple class that pulls together a few different pieces from the Router Module to surface them in a slightly easier API surface.

Constructor

new RouterWrapper()

Constructs a new RouterWrapper.

Instance Methods

this.registerRoute(capture, handler)

Parameters
capture String | Regex | Route

the The capture for a route can be one of three types.

1. It can be an Express style route, like: '/example/:anything/route/'
   The only gotcha with this is that it will only capture URL's on your
   origin.
2. A regex that will be tested against request URL's.
3. A Route object
handler function | Handler

The handler is ignored if you pass in a Route, otherwise it's required. The handler will be called when the route is caught by the capture criteria.
