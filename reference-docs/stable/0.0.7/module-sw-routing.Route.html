Route

Constructor

new Route(match, handler, method)

The constructor for Route expects an object with match and handler properties which should both be functions.

Parameters
match function

The function that determines whether the route matches.
handler Object

An Object with a handle method. That method will be used to respond to matching requests.
method string (Optional)

Only match requests that use this HTTP method. Defaults to 'GET' if not specified.
