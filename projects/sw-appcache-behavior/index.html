legacyAppCacheBehavior

legacyAppCacheBehavior(event)

The main entry point into the App Cache Behavior library. It is meant to be called by a service worker's fetch event handler:

self.addEventListener('fetch', event => {
  event.respondWith(legacyAppCacheBehavior(event).catch(error => {
    // Fallback behavior goes here, e.g. return fetch(event.request);
  }));
});

legacyAppCacheBehavior() can be selectively applied to only a subset of requests, to aid in the migration off of App Cache and onto a more robust service worker implementation:

self.addEventListener('fetch', event => {
  if (event.request.url.match(/legacyRegex/)) {
    event.respondWith(legacyAppCacheBehavior(event));
  } else {
    event.respondWith(robustServiceWorkerBehavior(event));
  }
});
